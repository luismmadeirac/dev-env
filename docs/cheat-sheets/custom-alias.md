# Custom Alias

## List

| COMMAND | DESCRIPTION |
| --- | --- |
| `d` | Docker |
| `k` | kubectl |
| `h` | Helm |
| `tf` | Terraform |
| `a` | Ansible |
| `t` | Tmux |
| `v` | Vim |

## Command to run custom scripts

| COMMAND | DESCRIPTION |
| --- | --- |
| `wifi` | On/Off mac os wifi |
| `kp` | Kill Process|
| `wh` | Outputs to the terminal timezone hours predefined, weird but useful |
| `password` | Outputs a new generated password |
| `ssh-gen` | Generates new SSH Key  |

## Navigation Commands

| COMMAND | DESCRIPTION |
| --- | --- |
| `wdir` | Go to Work dir |
| `pdir` | Go to Personal Projects dir |
